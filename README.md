# keyboard_receiver
0-9 - control the moving speed of turtle

space - stop

w - moving forward

d - moving backward

other keys - control the radius of circle


## how to use:

First catkin_make.

### Terminal 1:
roscore

### Terminal 2:
rosrun turtlesim turtlesim_node

### Terminal 3:
rosrun keyboard_receiver keyboard_receiver_node

### Terminal 4:
rosrun keyboard_receiver keyboard
